Favourite Tools

These are the tools I use to design, build, and refine modern, responsive websites.

HTML and CSS — Structure & Styling

The core tools behind every website I build. I use HTML to structure content properly and CSS to create clean layouts, responsive designs, and visually appealing interfaces across all screen sizes.

JavaScript — Interactivity & Functionality

I am still learning but I use JavaScript to add interactivity and dynamic features to websites. From simple animations to complex user interactions, JavaScript helps enhance the user experience and make websites more engaging.

Visual Studio Code — Development Environment

My primary code editor for building front-end projects. VS Code helps me write clean code, stay organized, and work efficiently with extensions and debugging tools.

Git & GitHub — Version Control

I use Git and GitHub to manage my projects, track changes, and push my work online. It helps me work professionally and keep my code safe and organized.

An essential tool for testing responsiveness, fixing layout issues, and inspecting elements. It allows me to fine-tune designs and ensure websites work well on different devices.
